## 2461. Maximum Sum of Distinct Subarrays With Length K

- You are given an integer array nums and an integer k. Find the maximum subarray sum of all the subarrays of nums that meet the following conditions:
- The length of the subarray is k, and All the elements of the subarray are distinct.
- Return the maximum subarray sum of all the subarrays that meet the conditions. If no subarray meets the conditions, return 0.
- A subarray is a contiguous non-empty sequence of elements within an array.


### Approach

- Use two pointer approach, and sliding window together, where the left and right are the lft and right ends of the window respectively.
- Use HashSet to solve this problem, because the problem asked us to find sum of distinct subarrays with length k.
- We use the `left` index to eliminate the left side elements as we move forward, and the right index to add new elements.
- So, while adding, if nums[right] already exists in the set, then remove nums[left] from the set also subtract it from the sum, otherwise add it to the set and update the sum
- In this way continue doing it for the rest of the numbers in the array.


```java
for (int right = 0; right < n; right++) {
            while (set.contains(nums[right])) {               
                sum -= nums[left];
                set.remove(nums[left]);
                left++;
            }

            set.add(nums[right]);
            sum += nums[right];

            if (right - left + 1 == k) {
                maxSum = Math.max(maxSum, sum);

                sum -= nums[left];
                set.remove(nums[left]);
                left++;
            }
        }
```
