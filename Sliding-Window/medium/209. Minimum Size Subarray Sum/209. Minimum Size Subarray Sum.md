## 209. Minimum Size Subarray Sum

- Given an array of positive integers nums and a positive integer target, return the minimal length of a subarray whose sum is greater than or equal to target. If there is no such subarray, return 0 instead.

```
Input: target = 7, nums = [2,3,1,2,4,3]
Output: 2
Explanation: The subarray [4,3] has the minimal length under the problem constraint.
```


### Approach

- This is a variable size sliding window problem.
- We will start with calculating the sum from the very first element and keep of adding elements.
- While the sum is greater than the target value, we remove the left end elements from the window and add the new elements.
- Update the count as you move forward, and also track the mincount.


```java
int left = 0;
        int sum = 0;
        int minCount = Integer.MAX_VALUE;
        for (int right = 0; right < n; right++) {
            sum += nums[right];
            while (sum >= target) {
                count = right - left + 1;
                minCount = Math.min(count, minCount);

                sum -= nums[left];
                left++;
            }
        }
        return minCount == Integer.MAX_VALUE ? 0 : minCount;
```
