## 1343. Number of Sub-arrays of Size K and Average Greater than or Equal to Threshold

- Given an array of integers arr and two integers k and threshold, return the number of sub-arrays of size k and average greater than or equal to threshold.

**Eg:**

```
Input: arr = [2,2,2,2,5,5,5,8], k = 3, threshold = 4
Output: 3
Explanation: Sub-arrays [2,5,5],[5,5,5] and [5,5,8] have averages 4, 5 and 6 respectively. All other sub-arrays of size 3 have averages less than 4 (the threshold).
```


### Approach

- This is a fixed size window type problem.
- First consider the very first window, from 0 to k-1, count its sum and average.
- If avg >= threshold, update the count value.
- Now slide the window one number at a time to the right, and update the sum accordingly and also calculate avg.
- Now, again if the avg >= threshold, update count every time.
- For updating the sum, subtract the number leaving the window from the sum and add the element entering the window.



### Time and Space Complexities

- **Time Complexity** : O(n)
- **Space Complexity** : O(1)
