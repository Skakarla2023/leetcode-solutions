## 1456. Maximum Number of Vowels in a Substring of Given Length

- Given a string s and an integer k, return the maximum number of vowel letters in any substring of s with length k.
- Vowel letters in English are 'a', 'e', 'i', 'o', and 'u'.

**Eg:**

```
Input: s = "abciiidef", k = 3
Output: 3
Explanation: The substring "iii" contains 3 vowel letters.
```


### Approach

- First take the very first window with size k, starting from 0, and count the vowels, and update  the count value.
- Now start with the next window, slide the window to the right, and update the count value by the following way:
  - if the character leaving the window is a vowel, `count--`.
  - if the character entering the window is a vowel, `count++`.
 


```java
for (int i = k; i < n; i++) {
            if (isVowel(s.charAt(i - k))) {
                count--;
            }

            if (isVowel(s.charAt(i))) {
                count++;
            }
            maxCount = Math.max(count, maxCount);
        }
```



### Time and Space Complexity

- **Time Complexity** : O(n)
- **Space Complexity** : O(1)
