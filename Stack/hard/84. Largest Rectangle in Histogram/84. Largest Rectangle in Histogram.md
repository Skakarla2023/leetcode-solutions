## 84. Largest Rectangle in Histogram

- Given an array of integers heights representing the histogram's bar height where the width of each bar is 1, return the area of the largest rectangle in the histogram.

<img width="522" height="242" alt="image" src="https://github.com/user-attachments/assets/90872f8b-42db-42b3-9a7f-8222e4a48094" />


```
Input: heights = [2,1,5,6,2,3]
Output: 10
Explanation: The above is a histogram where width of each bar is 1.
The largest rectangle is shown in the red area, which has an area = 10 units.
```


### Approach:

- The idea is to find the nearest smaller element to the right and left of each element(height).
- Because the height of the rectangle depends on the shorter bar rather the longer bar at a particular location.
- For that we store the nearest smallest elements to its left in an array called left[] and the nearest smallest elements to its right in right[], which we can use to calculate the width of the largest rectangle we can form at that particular index.
- and the height will be the current height that we are standing on right now.
- We can calculate height, width and area using the following formulas:
  - height = heights[i]
  - width = right[i]-left[i]-1
  - area = height * width
  - maxArea = Math.max(area, maxArea)
