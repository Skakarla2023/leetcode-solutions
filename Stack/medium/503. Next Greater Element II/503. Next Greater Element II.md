## 503. Next Greater Element II

- Given a circular integer array nums (i.e., the next element of nums[nums.length - 1] is nums[0]), return the next greater number for every element in nums.
- The next greater number of a number x is the first greater number to its traversing-order next in the array, which means you could search circularly to find its next greater number. If it doesn't exist, return -1 for this number.


**Eg:**

```
Input: nums = [1,2,1]
Output: [2,-1,2]
Explanation: The first 1's next greater number is 2; 
The number 2 can't find next greater number. 
The second 1's next greater number needs to search circularly, which is also 2.
```


### Approach

- Use a stack to solve this problem.
- For each element you traverse through the array, you have to compare it with number at the top of the stack, if the current element is greater then the stack.top(), update res[stack.top()] to this current element, which means the current element is the next greater element fot the previous element.
- If no such element is found, pop until you find such element.
- If the stack is empty, push the element into the stack.


```java
for (int i = 0; i < 2*n; i++) {
            int cirInd = (i%n);

            while (!stack.isEmpty() && nums[cirInd] > nums[stack.peek()]) {
                int ind = stack.pop();
                ans[ind] = nums[cirInd];
            }
            stack.push(cirInd);
        }
```
