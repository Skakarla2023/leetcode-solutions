## 102. Binary Tree Level Order Traversal

- Given the root of a binary tree, return the level order traversal of its nodes' values. (i.e., from left to right, level by level).

<img width="277" height="302" alt="image" src="https://github.com/user-attachments/assets/8aa48579-2404-41bd-b45a-9d6dbe01baeb" />

```
Input: root = [3,9,20,null,null,15,7]
Output: [[3],[9,20],[15,7]]
```

### Approach

- Use an arraylist to store the resultant values(i.e., the level order traversal).
- Use a queue to store the nodes level by level.
- Start from the root, put it into the queue.
- And pop it out form the queue, and add it to the list.
- And do the same for its children(left and right elements), add them into the queue(all the elements from the same level).
- And when you are done with all the nodes from the same level, pop them out from the queue and put it into the list.
- Repeat it for the rest of levels until you are done with all the levels.


```java
while (!q.isEmpty()) {
            int size = q.size();
            List<Integer> sublist = new ArrayList<>();

            for (int i = 0; i < size; i++) {
                TreeNode curr = q.poll();
                sublist.add(curr.val);

                if (curr.left != null) {
                    q.add(curr.left);
                }

                if (curr.right != null) {
                    q.add(curr.right);
                }
            }
            list.add(sublist);

        }
```
