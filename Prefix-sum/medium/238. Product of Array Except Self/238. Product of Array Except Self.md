# 238. Product of Array Except Self

**Difficulty:** Medium  
**Tags:** Arrays, Prefix Sum  
**Link:** https://leetcode.com/problems/product-of-array-except-self/

---

## Problem Statement

Given an integer array `nums`, return an array where each `answer[i]` is the product of all elements except `nums[i]`. Must run in **O(n)** without division.

---

## Examples & Edge Cases

| Input | Output | Note |
|---|---|---|
| `[1, 2, 3, 4]` | `[24, 12, 8, 6]` | Basic case |
| `[1, 0, 3]` | `[0, 3, 0]` | Single zero |
| `[0, 0, 3]` | `[0, 0, 0]` | Two zeros → all zero |
| `[-1, -2, 3]` | `[-6, -3, 2]` | Negatives |

---

## Approach

**Brute Force (`bruteforce.java`):** For each index `i`, multiply all elements where `j != i`. Simple but O(n²).

**Optimal (`solution.java`) — Prefix × Suffix:**  
For each index, `answer[i] = product of all elements to the LEFT × product of all elements to the RIGHT`.
- Left pass (L→R): fill `answer[i]` with running left product.
- Right pass (R→L): multiply in running right product in-place.

---

## Complexity

| | Time | Space |
|---|---|---|
| Brute Force | O(n²) | O(1) |
| Optimal | O(n) | O(1) |

---

## Key Takeaway

When division is off the table, **prefix × suffix** gives you "everything except me" in two clean passes.
