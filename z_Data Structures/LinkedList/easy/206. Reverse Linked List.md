# 206. Reverse Linked List

- Given a LinkedList, we have to reverse the linkedlist and return the `head` of the new linkedlist.
- Given the head, of a singly linkedlist , reverse the list and return the head.

### Approach

- Use three indices namely `curr`, `prev` and `future`.
- where:

<img width="334" height="151" alt="image" src="https://github.com/user-attachments/assets/bedcfde7-c700-4dd9-aa6d-3e92c5670d18" />

```java

if(head == null) {
  return null;
}

prev = head;
curr = head.next;
prev.next = null;

while(curr != null) {
  future = curr.next;
  curr.next = prev;
  prev = curr;
  curr = future;
}
```

### Time and Space Complexity

- **Time Complexity** : O(n)
- **Space Complexity** : O(1)
