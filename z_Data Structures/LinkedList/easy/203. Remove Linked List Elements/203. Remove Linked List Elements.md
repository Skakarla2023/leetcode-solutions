## 203. Remove Linked List Elements

- Given a singly linkedlist and a value, we have to return the head of the list after removing the occurences of the given value.

<img width="782" height="222" alt="image" src="https://github.com/user-attachments/assets/81db6b7d-da61-4b8d-9a35-c039530ecc5f" />


- To solve this problem, we check repeatedly for every element if it points to the node containing the value, move its next a node forward.
- for that we have the following logic:

```java
if(node.next.val == val) {
  node.next = node.next.next;
} else {
  node = node.next;
}
```


- But here, if you observe, the traversal starts at first node, and we are checking if(node.next.val == val), here we are skipping the first node, but the value might be even present at the first node.
- So we should start traversing from the first node.
- For that we create a dummy node in the beginning of the list and use the same logic for checking if value exists.

```java
ListNode node = new ListNode(0);
        node.next = head;
        ListNode dummy = node;
```
