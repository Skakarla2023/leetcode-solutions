## 160. Intersection of Two Linked Lists

- Given the heads of two linked lists, find the node at which the two linked lists intersect.
- Return the intersection node if it exists; otherwise, return null.
- We use a two-pointer approach where one pointer starts at the head of the first list and the other starts at the head of the second list.
- Move both pointers one step at a time.
- When a pointer reaches the end of its list, redirect it to the head of the other list.
- By switching lists, both pointers traverse the same total distance (lengthA + lengthB).
- If the lists intersect, the pointers will eventually meet at the intersection node (same memory address).
- If the lists do not intersect, both pointers will eventually become null at the same time.
- Therefore, when ptr1 == ptr2, return that node (or null if no intersection exists).

```java

while(pA != pB) {
  pA = (pA == null) ? headB : pA.next;
  pB = (pB == null) ? headA : pB.next;
}

return pA;
```
