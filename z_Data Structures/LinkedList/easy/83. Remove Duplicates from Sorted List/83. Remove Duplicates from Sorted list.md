## 83. Remove Duplicates from Sorted List

- Given head of a singly linked list, you have to return the list by removing duplicates, by keeping the list sorted.
- You have to modify the list in place, not creating a new list.

### Approach

- Use a pointer `curr` to traverse the list and compare values.
- If a duplicate is found anywhere skip the element.
- When a duplicate is not found, just movoe forward and continue comparing.


### Time and Space Complexity

- **Time Complexity** : O(n)
- **Space Complaexity** : O(1)


### Algotrithm

- Start traversing the list from beginning.
- If a duplicate is found anywhere:
  - skip the duplicate value.
- else :
  - move the pointer forward.
 


### PseudoCode 

```java
if(curr.val != curr.next.val) {
  curr = curr.next;
} else {
  curr.next = curr.next.next;
}
```
