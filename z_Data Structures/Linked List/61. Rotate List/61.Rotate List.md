# 61. Rotate List


Given the head of a linked list, rotate the list to the right by k places.

<img width="712" height="302" alt="image" src="https://github.com/user-attachments/assets/9b040f95-e048-47a2-945e-89f009d3aef0" />


Input: head = [1,2,3,4,5], k = 2

Output: [4,5,1,2,3]


- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
