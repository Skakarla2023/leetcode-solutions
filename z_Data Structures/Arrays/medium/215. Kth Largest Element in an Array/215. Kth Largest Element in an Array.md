## 215. Kth Largest Element in an Array

- Given an unsorted array and a non zero integer k, you have to return the kth largest element in that array.
- And we should solve it without sorting the array.

**Eg:**

```
Input: nums = [3,2,1,5,6,4], k = 2
Output: 5
```

**Approach**:

- We can use MaxHeap(priority queue).
- We can store the elements in any order(most probably ascending order).
- And as long as the size is greater than k, we will poll elements from the queue.
- And the reamining element from the peek(in the queue) is the required answer.



**Logic**:


```java
PriorityQueue<Integer> minHeap = new PriorityQueue<>();

        for(int i:nums) {
            minHeap.add(i);
        }

        while(minHeap.size() > k) {
            minHeap.poll();
        }

        return minHeap.peek();
```
