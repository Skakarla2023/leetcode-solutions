## 101. Symmetric Tree

- Given the root of a binary tree, return true if the given binary tree is a symmetric tree else false.
- By symmetric, it means if we place a mirror at the root of the binary tree, the left half of the tree should look like the mirror image of the right half or vice versa.

The above tree is not a binary tree.


**Approach**:

- If there is a single root node and no other nodes, then return true, because a single node is always symmetric.
- If there is no node existing, return true.
- If one of the nodes in the left or right is null return false.
- And go on comparing the values in the left with the values at right.


```java
if (t1 == null && t2 == null) {
            return true;
        }

        if (t1 == null || t2 == null) {
            return false;
        }

        if (t1.val != t2.val) {
            return false;
        }

        return isMirror(t1.left, t2.right) && isMirror(t1.right, t2.left);
```
