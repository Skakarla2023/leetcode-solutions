## 94. Binary Tree Inorder Traversal

- Given the root of a binary tree, you have to return the inorder traversal of its node values.
- **Inorder traversal**: Visiting the left node first and then the root node and then the right node of it.
- Store the inorder traversal of the tree in a list and return the list.



**Logic**:

```java
private void inorder(List<Integer> list, TreeNode root) {
        if(root == null) {
            return;
        }

        inorder(list, root.left);
        list.add(root.val);
        inorder(list, root.right);
    }
```
