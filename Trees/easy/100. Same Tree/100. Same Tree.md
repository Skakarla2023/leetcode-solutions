## 100. Same Tree

- Given the heads of two binary trees p and q, you have to return if the given trees are same.
- By same, it means the structure and elements of the tree should be similar.

<img width="382" height="182" alt="image" src="https://github.com/user-attachments/assets/affec8cf-e3ae-4012-971a-86b7bfb03c4d" />

The above trees are not similar.



**Approach**:

- If the given roots of both the trees are null, return true.
- If any one one of them is null and the other is not, then return false.
- And repeat this process for both the left subtrees and right subtrees of both the trees.



```java
if(p == null && q == null) {
            return true;
        }

        if(p == null || q == null) {
            return false;
        }

        if(p.val != q.val) {
            return false;
        }

        return isSameTree(p.left, q.left) && isSameTree(p.right, q.right);
```
