**Constraints:**

- nums1.length == m + n
- nums2.length == n
- 0 <= m, n <= 200
- 1 <= m + n <= 200
- -109 <= nums1[i], nums2[j] <= 109

### Approach:

- Scan elements from the end of the array so that you do not skip unvisited elements.
- Use two pointer approach,, use three pointers i,j,k with i being at the end of first array, j being at the end of second array and k for modifying the nums1 elements.
- when found nums1[i]>nums2[j]: do nums1[k] = nums1[i]; and i--;
- else nums2[j]>nums1[i]: nums1[k]=nums1[j]; j--;
- k--
