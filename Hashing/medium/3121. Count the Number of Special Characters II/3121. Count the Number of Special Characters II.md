# 3121. Count the Number of Special Characters II

- Difficulty : medium
- Topics : HashTable



 
## Problem

- You are given a string word, and a letter is called **special** if it appears in both uppercase and lowercase in the string word and every lowercase occurence of c appears before the first uppercase occurence of c.
- You have to return the number of special characters or letters in the string word.


#### Eg:

**Example 1:** 
```
Input: word = "aaAbcBC"

Output: 3

Explanation:

The special characters are 'a', 'b', and 'c'.
```

**Example 2:**

```
Input: word = "abc"

Output: 0

Explanation:

There are no special characters in word.

There are no special characters in word.

```
**Example 3:**

```
Input: word = "AbBCab"

Output: 0

Explanation:

There are no special characters in word.
```

### Approach

- Use HashMap to store the characters and their indices as `key-value` pairs.
- We have to store the first uppercase occurence of ch and last lowercase occurence of ch.


### Time & Space Complexity

- **Time Complexity**: O(n)
- **Space Complexity**: O(n)
