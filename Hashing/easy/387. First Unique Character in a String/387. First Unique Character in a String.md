## 387. First Unique Character in a String

- Given a string s, find the first non-repeating character in it and return its index. If it does not exist, return -1.


```
Input: s = "leetcode"

Output: 0

Explanation:

The character 'l' at index 0 is the first character that does not occur at any other index.
```


### Approach

- Use a hashmap to store all the frequencies of the characters in the given string.
- Store all the letters and their frequencies in the hashset.
- Traverse through the string and for every letter, check its frequncy in the hashmap , and if it 1 return the index of the letter.
- If no such letter is found in the string, return -1 at  the end.


```java
        HashMap<Character, Integer> freq = new HashMap<>();
        for(int i=0;i<n;i++) {
            freq.put(s.charAt(i), freq.getOrDefault(s.charAt(i),0)+1);
        }

        for(int i=0;i<n;i++) {
            if(freq.get(s.charAt(i)) == 1) {
                return i;
            }
        }

        return -1;
```
