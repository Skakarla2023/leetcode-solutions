## 1207. Unique Number of Occurrences

- Given an array of integers arr, return true if the number of occurrences of each value in the array is unique or false otherwise.

**Eg:**

```
Input: arr = [1,2,2,1,1,3]
Output: true
Explanation: The value 1 has 3 occurrences, 2 has 2 and 3 has 1. No two values have the same number of occurrences.
```

### Approach

- Use a hashmap to store the frequencies of each element.
- Traverse through the hashmap, find if any frequncy value is repeated, if any repeated value is found, return false.
- If by the time you reach the end, you dont find any such, return true.
- Use a HashSet to store the frequencies and to detect if it has unique freuencies or not.
